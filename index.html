Product Idea

BOBO Shabu & Sushi has been designed to simplify the process of ordering from menu. Primary goal is to provide users with convenient access to book a table and order food. This application not only displays menu but also indicates the status of order, whether it’s being prepared or has been served.

Problem

On my last birthday, I wanted to have shabu for dinner. Unfortunately, I couldn’t find a table and had to wait at restuarant so long time. After finally getting a table and the menu, there wrer delays in placing order. And not all the items I ordered were brought to table. These exprience inspired me to create an application for users who, like me, love to enjey shabu dinner

Mobile Application

Target Audience

Users prefer to buy premium shabu that offers good value for its price. They enjoy dining with friends and family and prefer food that looks fancy. These users value fast ordering and efficient service.
